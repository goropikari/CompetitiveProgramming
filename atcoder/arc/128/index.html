ARC 128

  1. A - Gold and Silver

A - Gold and Silver

問題

iii 日目に金を銀に, jjj 日目に銀を金に交換すると金の量は Ai/AjA_i / A_jAi​/Aj​ 倍になる. ここで

AiAj=AiAi+1×Ai+1Ai+2×⋯×Aj−1Aj \frac{A_i}{A_j} = \frac{A_i}{A_{i+1}} \times \frac{A_{i+1}}{A_{i+2}} \times \cdots \times \frac{A_{j-1}}{A_j} Aj​Ai​​=Ai+1​Ai​​×Ai+2​Ai+1​​×⋯×Aj​Aj−1​​

であるから iii 日目に金を銀に, jjj 日目に銀を金に交換することは

  • Ai/Ai+1A_i / A_{i+1}Ai​/Ai+1​

    • iii 日目に金を銀に交換

    • i+1i+1i+1 日目に銀を金に交換

  • Ai+1/Ai+2A_{i+1} / A_{i+2}Ai+1​/Ai+2​

    • i+1i+1i+1 日目に金を銀に交換

    • i+2i+2i+2 日目に銀を金に交換

  • ...

  • Aj−1/AjA_{j-1} / A_jAj−1​/Aj​

    • j−1j-1j−1 日目に金を銀に交換

    • jjj 日目に銀を金に交換

することと等価である. 金の量を増やすには Ai>Ai+1A_i > A_{i+1}Ai​>Ai+1​ となるところで交換を行えばいい.

void solve() {
    int n;
    cin >> n;
    vector<int> A(n), change(n, 0);
    rep(i,n) cin >> A[i];
    rep(i,n-1) {
        if (A[i] > A[i+1]) {
            change[i] ^= 1;
            change[i+1] ^= 1;
        }
    }
    print(change);
}

XOR をとっているのは同じ日に金を銀に, 銀を金にという交換を連続する操作があった場合に 打ち消し合わせるため.
