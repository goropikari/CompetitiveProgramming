D - Sequence Query

問題

コンテスト後自力AC

multiset 1 つだけ使っても解けるが, 2つ使うと 2, 3番のクエリで同じ処理が使え楽になる.

xxx以下 を扱うときは要素を −1-1−1 倍して 以上 の問題に直したほうが扱いが楽になる.

2つの多重集合をそれぞれ S2,S3S_2, S_3S2​,S3​ とする

  • クエリ 1 のとき

    • S2S_2S2​ に −x-x−x を, S3S_3S3​ に xxx を追加する.

  • クエリ 2 のとき

    • S2S_2S2​ に対して −x-x−x で lower_bound を取る

    • k−1k-1k−1 個インクリメントした iterator が末端を超えたら −1-1−1, 範囲内なら -1 * (*it) を返す

  • クエリ 3 のとき

    • S3S_3S3​ に対して xxx で lower_bound を取る

    • k−1k-1k−1 個インクリメントした iterator が末端を超えたら −1-1−1, 範囲内なら *it を返す

void query(multiset<ll> &s, ll sign) {
    ll x, k;
    cin >> x >> k;
    auto it = s.lower_bound(sign * x);
    if (it == s.end()) {
        cout << -1 << endl;
        return;
    }

    int rem = k - 1;
    while (it != s.end() && rem) {
        it++;
        rem--;
    }
    if (it == s.end()) cout << -1 << endl;
    else cout << sign * (*it) << endl;
}

void solve() {
    int q;
    cin >> q;
    multiset<ll> s2, s3;

    rep(i,q) {
        ll t;
        cin >> t;
        if (t == 1) {
            ll x;
            cin >> x;
            s2.insert(-x);
            s3.insert(x);
        } else if (t == 2) {
            query(s2, -1);
        } else {
            query(s3, 1);
        }
    }
}

E - Putting Candies

問題

K≤lK \leq lK≤l のときは愚直にシミュレーションする.
