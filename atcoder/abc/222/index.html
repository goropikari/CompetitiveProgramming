D - Between Two Arrays

問題

dp[i][x]dp[i][x]dp[i][x]: iii 番目までの数が決まっており, ci=xc_i = xci​=x のときの場合の数とする.

ci=xc_i = xci​=x となるときの場合の数は, ci−1≤xc_{i-1} \leq xci−1​≤x となる場合の数であるから

dp[i][x]=∑k=ai−1xdp[i−1][k]\displaystyle dp[i][x] = \sum_{k = a_{i-1}}^x dp[i-1][k]dp[i][x]=k=ai−1​∑x​dp[i−1][k]

これを元に naive に実装すると以下のようになる.

vector<vector<ll>> dp(n, vector<ll>(MAX, 0));
rep2(x,a[0],b[0]+1) dp[0][x] = 1;
rep2(i,1,n) {
    rep2(x,a[i],b[i]+1) {
        rep2(k,a[i-1],x+1) {
            dp[i][x] += dp[i-1][k];
            dp[i][x] %= MOD;
        }
    }
}

ll ans = 0;
rep(i,MAX) {
    ans += dp[n-1][i];
    ans %= MOD;
}
cout << ans << endl;

しかしこれでは最大で O(N×30002)=O(30003)O(N \times 3000^2) = O(3000^3)O(N×30002)=O(30003) の計算量がかかり制限時間に待ち合わない. TLE 提出コード

ここで dp[i][x]=dp[i][x−1]+dp[i−1][x]dp[i][x] = dp[i][x-1] + dp[i-1][x]dp[i][x]=dp[i][x−1]+dp[i−1][x] ということに気づくと, iii 番目の計算では dp[i][ai]dp[i][a_i]dp[i][ai​] のときだけ for loop を回して, ai<x≤bia_i < x \leq b_iai​<x≤bi​ の範囲にある xxx のときは すでに計算した結果を使い回せるということに気づく. よって DP テーブルの更新は以下のようになる.

vector<vector<ll>> dp(n, vector<ll>(MAX, 0));
rep2(x,a[0],b[0]+1) dp[0][x] = 1;
rep2(i,1,n) {
    rep2(x,a[i],b[i]+1) {
        if (x == a[i]) {
            rep2(k,a[i-1],x+1) {
                dp[i][x] += dp[i-1][k];
                dp[i][x] %= MOD;
            }
        } else {
            dp[i][x] = (dp[i-1][x] + dp[i][x-1]) % MOD;
        }
    }
}

ll ans = 0;
rep(i,MAX) {
    ans += dp[n-1][i];
    ans %= MOD;
}
cout << ans << endl;
